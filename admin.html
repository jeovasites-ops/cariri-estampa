Guia de Gestão do Site

Tudo que você precisa para atualizar produtos, textos, imagens e contatos — sem precisar de programação.

Como Usar: Dê 2 cliques e pronto!

Use o arquivo INICIAR-SITE.bat para abrir o site. Ele inicia um servidor local e o site abre automaticamente no navegador. Assim você edita apenas os JSONs e as mudanças aparecem ao recarregar a página — sem precisar mexer em nenhum HTML.

1

Dê duplo clique em INICIAR-SITE.bat. O site abrirá no navegador automaticamente.

2

Edite o arquivo assets/data/ARQUIVO.json desejado, salve e recarregue a página (F5). Pronto!

📄 Páginas do Site

assets/data/home.json

Página Inicial

  • Banners: Troque as imagens e textos do carrossel do topo.
  • Bento Grid: Gerencie os quadros de categoria (Papelaria, Gráfica, etc.).
  • Suporta imagem diferente para celular e desktop.
Apenas editar o JSON
assets/data/produtos.json

Catálogo de Produtos

  • Adicione ou remova produtos na lista lista_produtos.
  • Destaque: Se "destaque": true, aparece na Home.
  • Preço: Use apenas números. Ex: 59.90
  • Badge: Texto do selo. Ex: "Novo", "-20% OFF"
  • Categorias válidas: Papelaria, Gráfica, Vestuário, Brindes.
Apenas editar o JSON
assets/data/banner-loja.json

Banner da Loja

  • banner.ativo: true mostra o banner, false oculta.
  • imagem_desktop: Foto horizontal (1200×350px ideal).
  • imagem_mobile: Foto vertical/quadrada (600×400px ideal).
  • link: URL de destino ao clicar. Use "#" para sem link.
  • cupom.ativo: true exibe a faixa com o cupom e o código clicável.
  • cupom.codigo: Código que o cliente copia. Ex: "CARIRI10"
Apenas editar o JSON
assets/data/sobre.json

Página Sobre

  • imagem_capa: Foto de fundo grande no topo da página.
  • foto_perfil: Foto redonda do responsável (selfie ou logo).
  • texto_paragrafos: Cada item é um parágrafo da história.
  • estatisticas: Altere os números e descrições.
  • diferenciais / valores: Missão, Visão e Valores.
Apenas editar o JSON
assets/data/contatos.json

Contatos e Redes

  • whatsapp: Apenas números. Ex: 5588999999999
  • instagram: Somente o @perfil.
  • email: Seu e-mail profissional.
  • endereco: Texto exibido no rodapé.
Apenas editar o JSON
assets/data/blog.json

Blog / Portal

  • O primeiro artigo da lista se torna o destaque maior.
  • resumo: Texto que aparece antes do "Ler mais".
  • tags: Categorias do artigo (exibidas como chips).
  • imagem: URL da capa do artigo.
Apenas editar o JSON
assets/data/plugins.json

Categorias da Loja

  • Edite a lista "categorias" para adicionar ou remover abas de filtro.
  • A primeira categoria deve ser sempre "Todos".
  • Os nomes aqui devem bater com os nomes nos produtos.
Apenas editar o JSON

🚀 Simplificando (Hospedagem)

Se o site estiver hospedado num servidor (Hostinger, cPanel, etc.)

Quando o site estiver online (com domínio ou hospedagem), você só precisa editar os arquivos JSON. Não precisará mais mexer no bloco window.__DADOS__ dentro dos arquivos HTML. O servidor entrega os JSONs automaticamente para o site.

💡 Dicas Essenciais

Como usar imagens

Hospede suas fotos no ImgBB.com (gratuito) e copie o link direto. Use sempre URLs que terminem em .jpg, .png ou .webp.

Escolhendo ícones

Para mudar um ícone, acesse a biblioteca Phosphor, clique no ícone e copie o nome da classe. Ex: ph-fill ph-heart

Biblioteca de Ícones

Regra de Ouro do JSON

Nunca esqueça as vírgulas e aspas. Se o site ficar em branco ou sem dados, abra o arquivo JSON num validador online: jsonlint.com

📋 Passo a Passo Rápido para Atualizar Conteúdo

1. Edite o JSON

Abra o arquivo em assets/data/ e altere o que desejar.

2. Copie os dados

Copie o conteúdo atualizado do JSON.

3. Cole no HTML

Abra o .html correspondente e substitua o bloco window.__NOME_DATA__.

4. Recarregue

Salve os arquivos e recarregue a página no navegador (F5).
